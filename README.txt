To Execute the Automated Regression Test
1. Make sure that Eclipse Java is already installed in your PC
2. Make sure that Java Software Development Kit (JDK) already installed in your PC
3. Unzip the ManuelUntalan.rar
4. Open the Eclipse Java IDE
5. Go to File > Import Projects from File System
6. Click on Directory and Select the Folder ManuelUntalan
7. Click Finish
8. Open the Java file
9. Right Click and Go to Run As > Java Application
